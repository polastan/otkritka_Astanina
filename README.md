# Happy Birthday App

Приложение для отправки поздравительной открытки с днём рождения.

## Описание
- Приложение отображает поздравление с днём рождения.
- Текст поздравления и подпись выравниваются по центру.
- Используется Jetpack Compose для UI.
- Легко редактировать сообщение и подпись.
